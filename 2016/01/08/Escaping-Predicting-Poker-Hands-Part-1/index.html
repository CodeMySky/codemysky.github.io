Escaping: Predicting Poker Hands, Part 1

  • predicting-poker-hands

While Tom, our hero, is traveling alone in the desert, he is caught by a mysterious tribe. They really like to play card games, so they proposed that, the way to set Tom free is to win a game from them. Tom does not know anything about their card games, but the tribe won’t tell him anything. So tom observe them quietly, from their random speaking he get the type of some of their poker hands. You must help Tom predict the poker hand using your big data skills to help Tom escape!

Data Set

According to Tom’s observation, on each card there is a wired mark and a number on it. Tom draw them down on papers, they are . Tom name these marks 1, 2, 3, 4.
And there seems to be 10 types of poker hands. So Tom named them 0 to 9. Tom write them down one line at a time, using two numbers represent one card, and the last number represent the type of poker hand.

An example of the data is:

1, 11, 2, 13, 4, 2, 3, 1, 12, 0

Here, (1, 11), (2, 13), (4, 2), (1, 12) represents five cards. And 0 represents one type of the card. All his records can be downloaded here.

After collecting so many records, Tom decide to do some basic exploration. He decide to explore the distribution of the data first. By observing data carefully, he is finally able to get some insight of the data and unveil the mystery of the card game.


According to the distribution, we can see that some of the types are extremely rare, but they may be very valuable. We want to avoid bias towards the majority.

Conclusion

Usually it is very hard to predict an open system, like stock market, while it is quite easy to predict the behavior of a closed system. Unfortunately, most of the closed systems are already quite well studied by scientist. Never the less, today we are exploring a closed system, a game. So let our journey begin.
