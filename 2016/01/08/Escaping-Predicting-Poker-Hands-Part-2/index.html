What are training set and test set and why are they important?

Training Set

When data scientists start their analyze process, they usually first split all data into two sets, training set and test set. The data used to discover potentially predictive relationships is called training set. Why do we need test set? Isn’t more data we have the better possibility to find more convincing relationships?

Let’s take an example here, if we want to estimate the gender ratio in all companies in the United States, but we don’t have enough money to go to every company. So we can randomly choose five companies to go to. If we have chosen Apple, Microsoft, Google, LinkedIn and Amazon, we can falsely make a conclusion that all companies in the United States are dominated by male engineers. But that is not true.

Test Set

So we need a data set which is independent from training set to evaluate the result of the training. We assume that test set has the same distribution as the training set.

How to split training set and test set? Usually we take about 30% of the total data set as test set. But while the data set is very large, we can take 5% or 3% to be the test set.
