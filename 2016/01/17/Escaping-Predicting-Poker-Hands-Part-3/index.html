Decision Tree

Random forest is a famous algorithm that has a good performance, and it is easy to explain how it works.
But before Tom introduce Random Forest to you ,he want first cover the concept of decision tree.

A decision tree is a tree-like model of decisions and their possible consequences. Here is a decision tree of how to distinguish products from Apple.

Sklearn provides an simple way to generate a decision tree

Random Forest

Decision tree is very sensitive to biased data, to avoid this bias, we can use multiple decision trees.
Random forest randomly choose a subset of all the features and build a decision tree upon it. With the increase of tree numbers, the biases will tend to balance.

Sklearn also provides an simple way to use random forest directly.

Baseline

So Tom decide to use random forest directly to generate a baseline.
